les notions de base

Git est un système de gestion de version distribué qui est largement utilisé pour suivre et gérer les modifications apportées à un projet logiciel. Voici quelques notions de base de Git : Dépôt (Repository) : Un dépôt est un répertoire qui contient l'historique complet de votre projet, y compris tous les fichiers et dossiers, ainsi que les informations sur les modifications qui ont été apportées. Il existe deux types de dépôts : les dépôts locaux (stockés sur votre propre machine) et les dépôts distants (stockés sur des serveurs, tels que GitHub, GitLab ou Bitbucket)
